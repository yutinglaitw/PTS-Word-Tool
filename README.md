# PTS Word Tool
公視節目《一字千金》查詢工具，使用辭典為《重編國語辭典 修訂本》與《成語典》

![](./screenshots/demo.jpg)

## 功能
- 《隻字片語》
   - 填詞查詢: 查詢符合指定條件的成語
   
- 《字字珠璣》
   - 隨機出題: 隨機出四字題目，支援一對四、四對一兩種題型
   - 答案匹配: 搜尋符合指定條件的字
